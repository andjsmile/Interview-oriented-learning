**插入一个知识点**

sort的顺序和 priority_queue的顺序刚好相反

```cpp
sort(a.begin(),a.end())   // 默认就是less<int>
// 0，1，2，3，4，5，6
sort(a.begin(),a.end(),greater<int>())
// 6,5,4,3,2,1,0

// 但是和优先队列还是有很大的不同

priority_queue<int> q;

// 6,5,4,3,2,1,0     默认就是最大顶点堆

priority_queue<int,vector<int>,greater<int>> q2;

// 0,1,2,3,4,5,6
```
