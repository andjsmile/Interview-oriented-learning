### 类型转换

- 子类转父类(向上转型)

- 父类转子类(向下转型)


引用类型和对象类型不一致的时候，才需要进行类型转换

判断转型能够成功:

把右边的当做左边来用(结构定义上说的通),就可以实现类型转换

子类转换为父类都是说的通的，


父类转换子类不一定能不能成功,
失败的表现形式是抛出异常 ClassCastException 类型转换异常

-  无继承关系的两个类，互相转换一定会错误


- 实现类转换为接口(向上转型)

- 接口转换成实现类(向下转型)
